Fast Olympic Coding plugin for Sublime Text 3 **Nathan's Version**

**基于[Jatana/FastOlympicCoding](https://github.com/Jatana/FastOlympicCoding)修改**

> - 修改了输出方式,现在**stdout**和**stderr**是分开显示的,同时展示**Expected Output**
> - (每个栏目前的高亮显示取决于编辑器当前配色)
> - 增加了监听器,在切换文件时会自动打开对应测试面板(测试文件存在的话)
> - 现在当测试面板打开时,焦点会保持在原文件,而不是测试面板
